# Project Overview and Purpose
Users often struggle to find songs that match their mood or activity. Navigating large music libraries is a
time-consuming process that reduces both productivity and overall user satisfaction. There exists a clear
need for an intelligent system capable of recommending music based on a user's emotional state in an
efficient and reliable manner. MoodBeats is a rule-based expert system designed to recommend songs in accordance with a user's
current mood and activity. Upon launching the system, users respond to a brief set of questions covering
their mood, current activity, energy level, and lyrical preference. The system then applies a set of
predefined IF–THEN rules that establish logical mappings between music attributes — including genre,
tempo, energy level, and lyrical content — and corresponding mood categories. The output is a tailored
playlist delivered through a simple, accessible user interface. The system is intended for music listeners who seek mood-based, context-aware playlist recommendations. Target use cases include studying, enhancing productivity, facilitating relaxation, and
supporting emotional regulation.
